FoodBridge – Smart Food Donation and Distribution Management System



Project Overview

FoodBridge is a food donation and distribution platform developed to help connect food donors with people or organizations in need. The system helps reduce food waste and supports efficient food sharing.

Features
- User Registration and Login
- Food Donation Management
- Food Request Management
- Donor and Receiver Tracking
- Food Availability Updates
- Simple and User-Friendly Interface

 Technologies Used
- Java
- Spring Boot
- Maven
- MySQL
- HTML
- CSS
- JavaScript
  Project Structure
- src/ - Source code
- pom.xml - Maven configuration
- mvnw / mvnw.cmd - Maven wrapper files

Purpose

The main objective of FoodBridge is to reduce food wastage by creating a platform where excess food can be donated and distributed to those who need it.
